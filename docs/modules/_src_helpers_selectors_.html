getNodeWithText

  • getNodeWithText(rootLocator: Locator, textoProcurado: string, textLocator?: Locator): Promise<any>
    • Defined in src/helpers/selectors.ts:92

    Retorna um [[ElementFinder]] correspondente ao nó passado no parametro rootLocator que contenha o mesmo texto que o passado em textoProcurado. É possível passar um complemento para o nó buscado, este de onde será extraído o texto para comparação, sendo assim é possível retornar um nó que não seja o qual o elemento com texto está localizado.

    Parameters

    • rootLocator: Locator

      nó raíz que será retornado

    • textoProcurado: string

      texto desejado a ser buscado

    • Optional textLocator: Locator

      caminho para onde estará localizado o texto

      suponha que há uma tabela de dados do tipo

      | nome | concluido |
| t1   | False     |
| t2   | True      |

      e cada linha da tabela é dada pelo xpath:

      const root_locator = By.xpath('//tbody//tr')

      e o xpath da coluna nome seja:

      // atenção com o '.' no inicio para indicar que é um subnó
nome_locator = By.xpath('.//td//span//span')

      e a opção de concluido seja dado pelo xpath:

      // atenção com o '.' no inicio para indicar que é um subnó
opt_concluido = By.xpath('.//td//input')

      Então, suponha que seja necessário alterar a coluna concluido com base no nome que for passado. Logo, a chamada para esta função seria:

      const row = getNodeWithText(root_locator, 't1', nome_locator)

      como row irá conter informação sobre toda a sua linha, pode-se fazer:

      // isto irá alterar o valor de concluído de t1 para True
await row.element(opt_concluido).click()

    Returns Promise<any>

selectFrom

  • selectFrom(locator: Locator, opcaoProcurada: string): Promise<void>
    • Defined in src/helpers/selectors.ts:30

    Dada uma lista de opções (com caminho já apontando para os nós que conterão o texto), seleciona a opção que contenha o mesmo texto que o procurado

    suponha que há um campo <select> do tipo

    espécie:

    • Aedes aegypti
    • Aedes albopictus
    • Anopheles

    e cada texto da opção é dada pelo xpath:

    const locator = By.xpath('//select//option//span')

    e deseja-se selecionar a opção que contém o texto 'Anopheles', então a chamada para a função fica:

    await selectFrom(locator, 'Anopheles');

    Parameters

    • locator: Locator

      caminho para um ou mais nós que possuam um texto dentro para ser comparado

    • opcaoProcurada: string

      string que deverá ser comparada

    Returns Promise<void>
